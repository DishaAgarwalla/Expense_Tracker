# 🚀  Expense Tracker

🌐 **Live Demo:** [View Application](https://expense-tracker-five-sable.vercel.app/)

A powerful, feature-rich expense tracking application built with React. Perfect for managing personal finances with beautiful visualizations and insights.

---

## ✨ Features

### 🔐 User Authentication

* Secure login and registration
* Personalized expense tracking per user
* Session management

### 💰 Expense Management

* Add, edit, delete expenses
* Multiple categories with icons
* Add notes and dates to expenses
* Bulk delete operations

### 📊 Advanced Analytics

* Interactive pie charts for category distribution
* Monthly bar charts for spending patterns
* Trend line charts for expense over time
* Budget tracking with progress bars
* Category-wise breakdown

### 🎨 Modern UI/UX

* Dark/Light theme toggle
* Responsive design for all devices
* Smooth animations and transitions
* Beautiful gradient effects
* Loading skeletons

### 📁 Data Management

* Export to CSV, Excel, PDF
* Local storage persistence
* Search and filter expenses
* Sort by multiple criteria
* Date range filtering

### 🏗️ Advanced Architecture

* Context API for state management
* Custom hooks for reusable logic
* Modular component structure
* Utility functions for helpers
* Environment configuration

---

## 🛠️ Tech Stack

* **Frontend**: React 18, React Router 6
* **State Management**: Context API + useReducer
* **Styling**: CSS3 with CSS Variables
* **Charts**: Recharts
* **Export**: XLSX, jsPDF
* **Icons**: React Icons
* **Build Tool**: Vite
* **Code Quality**: ESLint


## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/DishaAgarwalla/ExpenseTracker.git

# Navigate to project directory
cd ExpenseTracker

# Install dependencies
npm install

# Start development server
npm run dev
```
